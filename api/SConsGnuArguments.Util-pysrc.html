 34      """Map all predefined GNU variable triples (name, desc, default) via 
 35      `callback`. 
 36   
 37      :Parameters: 
 38          callback : callable 
 39              function of type ``callback(name, desc, default)``, where 
 40   
 41                  - ``name:`` is the name of variable being processed, 
 42                  - ``desc:`` is short description, 
 43                  - ``default:`` is the default value for the variable. 
 44          triples : list 
 45              a list of 3-element tuples with, each entry of the list should 
 46              be a tuple of the form ``(name, desc, default)``, where ``name`` 
 47              is the name of argument, ``desc`` is description (used as help 
 48              message) and ``default`` is default value. 
 49          name_filter : callable 
 50              callable object (e.g. lambda) of type ``name_filter(name) -> 
 51              boolean`` used to filter-out unwanted variables; only these 
 52              variables are processed, for which name_filter returns ``True`` 
 53   
 54      :Returns: 
 55          returns result of mapping through `callback` 
 56      """ 
 57      if SCons.Util.is_Sequence(name_filter): 
 58          seq = name_filter 
 59          name_filter = lambda x : x in seq 
 60      triples = filter(lambda t : name_filter(t[0]), triples) 
 61      return map(lambda x : callback(*x), triples)
