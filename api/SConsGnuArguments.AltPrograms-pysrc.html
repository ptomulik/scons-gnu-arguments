  1  """`SConsGnuArguments.AltPrograms` 
  2   
  3  SCons *arguments* for `Alternative Programs`_. 
  4   
  5  **General Description** 
  6   
  7  This module provides standard `autoconf output variables`_  which define paths 
  8  to `Alternative Programs`_. Each variable may be accessed via: 
  9   
 10      - SCons environment, as construction variables (``env.subst('$variable')``), 
 11      - SCons command line variables (``scons variable=value``), 
 12   
 13  Supported variables: 
 14  ==================== 
 15   
 16      AWK 
 17          The awk program 
 18      EGREP 
 19          The egrep program 
 20      FGREP 
 21          The fgrep program 
 22      GREP 
 23          The grep program 
 24      INSTALL 
 25          The program used to install files 
 26      INSTALL_DATA 
 27          The program used to install data 
 28      INSTALL_PROGRAM 
 29          The program used to install programs 
 30      INSTALL_SCRIPT 
 31          The program used to install scripts 
 32      LEX 
 33          The lex program 
 34      LEX_OUTPUT_ROOT 
 35          The base of the file name that the LEX generates 
 36      LEXLIB 
 37          Library that should be linked to LEX-generated program 
 38      LN_S 
 39          Either 'ln -s', 'cp -pR' or just 'ln' 
 40      MKDIR_P 
 41          Either 'mkdir -p' or 'install-sh' 
 42      RANLIB 
 43          The ranlib program 
 44      SED 
 45          The sed program 
 46      YACC 
 47          The yacc program 
 48   
 49  .. _autoconf output variables: http://www.gnu.org/software/autoconf/manual/autoconf.html#Output-Variable-Index 
 50  .. _Alternative Programs: http://www.gnu.org/software/autoconf/manual/autoconf.html#Alternative-Programs
